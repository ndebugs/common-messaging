Attribute / Required

Indicates the Field is required.

Properties:
emptyAllowed Boolean value. Indicates this Required is allowed empty value or not.
value Boolean Lang statement. Indicates the Field is required or not.

If value is TRUE and emptyAllowed is FALSE, then Message Value must be not empty.
If both value and emptyAllowed is TRUE, then Message Value must be not NULL.
If condition did not match, then FieldRequiredException will be thrown.
