Attribute / Length

Limit Message Value's size.

Properties:
fixed Boolean value, indicates whether this Length is fixed or not.
minValue Unsigned Integer, indicates minimum length. Used only if fixed is TRUE
value Signed Integer, indicates maximum length. -1 means unlimited.

Fixed length means Message Value's size must equals with value.
If condition did not match, then LengthValidationException will be thrown.
